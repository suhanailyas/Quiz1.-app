## 📌 Project Description

This project is a responsive and interactive Quiz Application developed using HTML, CSS, JavaScript, and Bootstrap. The application allows users to attempt multiple-choice questions with a timer and view their results at the end.

---

## ✨ Features

* 🎮 Start screen (Name & Registration Number input)
* ❓ Multiple-choice questions (A, B, C, D)
* 🔀 Randomized questions
* ⏱ Countdown timer for each question
* 📊 Multi-color progress bar
* 🎨 Dark/Light mode toggle
* ✅ Highlight correct and incorrect answers
* 📄 Result screen with answer review
* 📥 Download result as PDF
* 📱 Responsive design

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the folder
3. Open `index.html` in browser
